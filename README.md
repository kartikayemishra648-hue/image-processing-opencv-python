# image-processing-opencv-python
This repository contains implementations of fundamental image processing operations using OpenCV in Python. It includes pixel manipulation, ROI (Region of Interest) selection, image cropping, border creation, channel splitting and merging, arithmetic operations, image blending, and bitwise operations.
Here’s a **2-paragraph brief description** (simple + clean):

> This repository contains basic image processing operations implemented using OpenCV in Python. It covers essential concepts like pixel access and manipulation, region of interest (ROI), image cropping, and understanding image properties such as shape and size. These examples help in building a strong foundation in computer vision.

> It also includes advanced operations like image addition, subtraction, blending, bitwise operations, border creation, and channel splitting and merging. The project is designed for beginners to learn OpenCV practically through simple and clear code examples.
